What is Bloggee?

This Content Management System is suitable for you if you are planning to have a small and simple personal blog with relatively small ammounts of blog post contents. If you want to make an extensively large content website, then you better to find any other CMS.

Getting Started

  • Download bloggee
  • Delete data.txt file
  • Open config.php
  • Set your Admin Username and Password
  • Change the data.txt to something that others cannot guess like 617eff5e852b8.txt
  • Login to admin panel: yoursite/admin.php
  • Start creating!
